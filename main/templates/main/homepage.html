How to make a set

Quizlet:

1. Save a quizlet as your own
2. Click the 3 dots and click 'Export'
3. Set between to 'tab' and between rows to 'new line'
4. Click this link and paste it!

Google Sheets:

1. Put the front of each card under the 'A' column
2. Put the back of each card under the 'B' column
3. Click File > Download > Tab Seperated Values (.tsv)
4. Click this link and upload it!
